## Vue와 React의 공통점
둘 다 JS 기반

Vue : 컴포넌트 기반의 SPA를 구축하게 해주는 프레임워크

SPA : 하나의 페이지 안에서 필요한 영역만 로딩되는 형태 (ajax 후속작)


-----------------------------------------------------------------------------------

## Vue와 React의 차이점
Vue는  HTML, JS, CSS를 구분해서 작성 (프레임워크)

React는 JS하나로 DOM과 UI를 다 구성함 (UI라이브러리) 

그래서 Vue가 훨씬 작성하기는 편하다
